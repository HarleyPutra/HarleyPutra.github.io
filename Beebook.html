BeeBook

Beebook is a book reading website made using Laravel 10 that is made by a team comprised of Harley Putradinata, Bryan Januardo, Ferry Febrian, Nicholas Raven Shane, and Wilson Wijaya. beebook is a book reading website in development meant for indonesian students to easily access free or cheap books to read and study. It has a reward system based on how much the user has read and the rewards that the user receive is a voucher for a book of their choice. Laravel 10

This project was developed using Laravel 10, a powerful PHP framework known for its elegant syntax and robust features. The application leverages Laravel's built-in tools for routing, authentication, and database management, ensuring a streamlined development process.

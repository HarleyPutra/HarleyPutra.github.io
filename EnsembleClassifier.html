Leukemia Ensemble Classifier

A group project made by me and Bryan Januardo, this ensemble classifier was made using the Python programming language, made to classify Acute Lymphoblastic Leukemia. And later published for the 9th International Conference on Computer Science and Computational Intelligence.

Python Programming Language

This project was developed using the Python programming language, a versatile and easy to use programming language.

Support Vector Machine

This project project utilized Support Vector Machines (SVM) to classify data and identify patterns within the dataset.

Published on 9th ICCSCI

This project was further developed into a research paper, and was subsequently published and presented in the 9th International Conference on Computer Science and Computational Intelligence.
